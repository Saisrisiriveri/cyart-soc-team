**Capstone Project: Full Alert-to-Response Cycle**

**Overview**

This project simulates a real-world security incident from detection to response.

**Attack Simulation**

A brute-force SSH attack was simulated using repeated login attempts.

Detection

The system generated an alert for multiple failed login attempts.

Triage

The alert was analyzed and classified as Medium priority.

**Response**

- Identified attacker IP address
- Blocked malicious IP
- Monitored system logs

**Reporting**

An incident report was created documenting the event.

**Conclusion**

The attack was successfully detected and mitigated.
