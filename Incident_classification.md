**Incident Classification**

**Overview**

Incident classification helps categorize security events for better analysis and response.

**Examples:**

- Check SIEM alerts (Splunk / ELK)
- Analyze logs (IP, timestamps, user activity)
- Validate false positive vs true incident

**Types of Incidents**

| **Incident Type** | **Description** | **MITRE ATT&CK ID** |
| --- | --- | --- |
| Phishing | Fake emails to steal data | T1566 |
| Malware | Malicious software infection | T1204 |
| Data Exfiltration | Unauthorized data transfer | T1041 |
| DDoS Attack | Flooding network traffic | T1498 |

**Explanation**

- **Phishing:** Attackers trick users through fake emails
- **Malware:** Software that damages systems
- **Exfiltration:** Data stolen from systems

**Conclusion**

Classification improves investigation efficiency.
